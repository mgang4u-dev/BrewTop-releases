# BrewTop

**macOS용 Homebrew GUI 패키지 관리자**

터미널 없이 Homebrew 패키지를 설치, 삭제, 업데이트할 수 있는 네이티브 macOS 앱입니다.

## Features

- Formulae (CLI 도구) 및 Cask (GUI 앱) 관리
- 패키지 검색 및 상세 정보 확인
- 업데이트 가능 패키지 확인 및 일괄/개별 업데이트
- Tap 관리 (추가/삭제)
- Cleanup (캐시/이전 버전 정리) 및 Doctor (시스템 진단)
- 다국어 지원 (English, 한국어, 日本語)

## Download

[**Latest Release**](https://github.com/mgang4u-dev/BrewTop-releases/releases/latest)에서 DMG 파일을 다운로드하세요.

## Install

1. DMG 파일을 열고 `BrewTop.app`을 `/Applications`로 드래그
2. 앱 실행 (Homebrew가 설치되어 있어야 합니다)

## Requirements

- macOS 13.0 (Ventura) 이상
- Apple Silicon (arm64)
- [Homebrew](https://brew.sh) 설치 필요
